# Sorting-Algorithms-Visualizer-Console-Based-c-programe-\\

1.How to Start the Program\
Compile the source code using a C++ compiler.\
Run the executable file from the terminal or command prompt.\
The main menu will appear on the screen.\

2. Menu Options\
The user is typically provided with the following options:\
Select a sorting algorithm (Bubble, Insertion, Selection, Merge, Quick).\
Choose the size of the array.\
Choose the input type (random, sorted, reverse sorted).\
Start the visualization process.\
Exit the program.\
